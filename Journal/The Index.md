# The Index

> [!abstract]
> - 索引讓你在寫筆記的數天、數月、數年之後，仍能輕鬆地找到當初的筆記內容。
> - 在數位筆記中，我將其視為一切筆記、索引筆記的入口

- [[Future Log]] - 有明確的日期的事件
- [[Backlog]] - 沒有明確日期的待辦事項清單
